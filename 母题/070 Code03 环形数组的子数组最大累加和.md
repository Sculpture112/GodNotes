**核心套路:** 

“破环为直”与“正难则反”。环形数组的最大子数组和只可能有两种形态：

1. **不跨界（常规情况）：** 完全在原数组内部，直接用经典 Kadane 算法求【全局最大子段和】(`maxsum`)。
    
2. **跨界（环形情况）：** 包含首尾部分，相当于原数组的总和挖去了中间的一段。要使剩下的两端之和最大，被挖去的中间一段必须是【全局最小子段和】(`minsum`)。此时最大和为 `all - minsum`。 **边界防御：** 当数组全为负数时，`minsum` 会等于 `all`，导致 `all - minsum = 0`（相当于什么都不选，也就是空数组）。题目要求子数组非空，因此这种情况必须直接返回 `maxsum`（即数组中最大的那个负数）。


```cpp
// 070 Code03 环形数组的子数组最大累加和

// 给定一个数组nums，长度为n
// nums是一个环形数组，下标0和下标n-1是连在一起的
// 返回环形数组中，子数组最大累加和
// 测试链接 : https://leetcode.cn/problems/maximum-sum-circular-subarray/

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

class Solution {
public:
    int maxSubarraySumCircular(vector<int>& nums) {
        int n = nums.size();
        if (n == 0) return 0;
        
        int all = nums[0];
        int maxsum = nums[0];
        int minsum = nums[0];
        int maxpre = nums[0];
        int minpre = nums[0];
        
        for (int i = 1; i < n; i++) {
            all += nums[i];
            
            // 维护当前位置结尾的最大子数组和
            maxpre = max(nums[i], nums[i] + maxpre);
            maxsum = max(maxsum, maxpre);
            
            // 维护当前位置结尾的最小子数组和
            minpre = min(nums[i], nums[i] + minpre);
            minsum = min(minsum, minpre);
        }
        
        // 1) maxsum：最大子段和没有跨越首尾边界
        // 2) all - minsum：最大子段和跨越了首尾边界（即挖去了中间的最小子段）
        // 特殊情况：如果所有数都是负数，all == minsum，此时 all - minsum = 0
        // 但我们不能选空数组，所以必须返回那个绝对值最小的负数，即 maxsum
        return all == minsum ? maxsum : max(maxsum, all - minsum);
    }
};
```
