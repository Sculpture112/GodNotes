**核心套路:** 
乘法运算与加法运算最大的不同在于“负负得正”**。这意味着乘积的单调性会被负数打破：前一刻极小的负数（局部最小值），遇到一个新的负数后，会瞬间翻转成极大的正数（局部最大值）。 因此，在遍历计算以 $i$ 结尾的子数组乘积时，必须**同时维护两个状态：

1. `cur_max`：以当前元素结尾的连续子数组的**最大乘积**（尽量大的正数，用于应对下一个数是正数的情况）。
    
2. `cur_min`：以当前元素结尾的连续子数组的**最小乘积**（尽量小的负数，用于应对下一个数是负数的情况，期盼负负得正）。
    

当遇到 $0$ 时，乘积归零，相当于强制在这个位置“断开”，重新开始寻找新的子数组。核心转移方程就是在 `num`, `cur_max * num`, `cur_min * num` 这三者中寻找新的极值。


```cpp
// 071 Code01 乘积最大子数组

// 给你一个整数数组 nums
// 请你找出数组中乘积最大的非空连续子数组
// 并返回该子数组所对应的乘积
// 测试链接 : https://leetcode.cn/problems/maximum-product-subarray/

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

class Solution {
public:
    int maxProduct(const vector<int>& nums) {
        if (nums.empty()) return 0;

        // 这节课讲完之后，测试数据又增加了
        // 用 int 或 long long 类型的变量可能会让中间结果溢出
        // 所以改成用 double 类型的变量
        // 思路是不变的
        double ans = nums[0];
        double cur_min = nums[0];
        double cur_max = nums[0];

        for (int i = 1; i < nums.size(); i++) {
            double num = nums[i];
            
            // 使用 C++11 初始化列表一次性求出三个数的最值
            double next_min = min({num, cur_min * num, cur_max * num});
            double next_max = max({num, cur_min * num, cur_max * num});
            
            cur_min = next_min;
            cur_max = next_max;
            ans = max(ans, cur_max);
        }
        
        return (int)ans;
    }
};
```
