// 059 Code05 戳印序列
// 测试链接 : https://leetcode.cn/problems/stamping-the-sequence/
// 核心思路：逆向思考 + 拓扑排序

#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

class Solution {
public:
    vector<int> movesToStamp(string stamp, string target) {
        int m = stamp.length();
        int n = target.length();
        int numPositions = n - m + 1;
        
        vector<int> indegree(numPositions, m);
        vector<vector<int>> graph(n); // 每个字符位置被哪些印章起始点依赖
        vector<int> q(numPositions);
        int l = 0, r = 0;

        // 1. 预处理建图和入度
        for (int i = 0; i < numPositions; ++i) {
            for (int j = 0; j < m; ++j) {
                if (target[i + j] == stamp[j]) {
                    if (--indegree[i] == 0) q[r++] = i;
                } else {
                    graph[i + j].push_back(i);
                }
            }
        }

        // 2. 拓扑排序（逆向模拟）
		vector<bool> visited(n, false);
		vector<int> path;
		while (l < r) {
			int cur = q[l++];
			path.push_back(cur);
			// 盖下起始于 cur 的印章，影响 cur 到 cur + m - 1
			for (int i = 0; i < m; ++i) {
				int pos = cur + i;
				if (!visited[pos]) {
					visited[pos] = true;
					for (int nextIdx : graph[pos]) {
						if (--indegree[nextIdx] == 0) {
							q[r++] = nextIdx;
						}
					}
				}
			}
		}

        // 3. 验证并逆序返回
        if (path.size() != numPositions) return {};
        reverse(path.begin(), path.end());
        return path;
    }
};
