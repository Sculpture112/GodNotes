# []

> **原题链接:** ()

**涉及知识点:** [[01背包]], [[多重背包]]，[[需重刷]],[[二进制分组]],[[已AC]]

**核心套路:** 
1. **降维转化:** 多重背包（有限数量）本质上是 01背包的 Pro Max 版。核心思想是把同种类的多个物品“打包”，变成多个完全独立的 01 背包物品。
2. **二进制倍增打包 (核心性能优化):** 绝不能 $1,1,1...$ 地老实拆分（会导致超时）。利用 $2^0, 2^1, 2^2 \dots 2^k$ 以及最后剩余常数的组合，可以拼凑出 $0 \sim cnt$ 的任何数。这样拆分复杂度从 $O(cnt)$ 暴降至 $O(\log cnt)$。 
3. **滚动数组 (一维DP):** 拆完之后，直接套用标准 01 背包一维模版。外层正序遍历“超级包裹”，内层**逆序**遍历容量。

// 075 Code02 多重背包通过二进制分组转化成 01 背包 (模版)

// 宝物筛选
// 一共有 n 种货物, 背包容量为 t
// 每种货物的价值(v[i])、重量(w[i])、数量(c[i])都给出
// 请返回选择货物不超过背包容量的情况下，能得到的最大的价值
// 测试链接 : https://www.luogu.com.cn/problem/P1776

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

/**
 * 💡 核心逻辑：二进制分组 (Binary Splitting)
 * 
 * 为什么要分组？
 * 直接枚举每种物品选几个太慢了（O(n * t * avg_cnt)）。
 * 
 * 怎么分组？
 * 任何正整数 cnt 都可以拆成一系列 2 的幂次之和：1, 2, 4, 8... 以及一个剩余量。
 * 比如 cnt = 13:
 * 1. 拿走 1，剩 12
 * 2. 拿走 2，剩 10
 * 3. 拿走 4，剩 6
 * 4. 下一个幂次是 8，但 6 < 8，所以剩下的 6 直接作为一个整体。
 * 最终 13 被拆为：1, 2, 4, 6。
 * 
 * 用这四个“新物品”进行 01 背包，可以组合出 0~13 之间的任何数量，且绝不会超过 13。
 * 这样我们就把多重背包转化成了 01 背包，复杂度从 O(cnt) 降到了 O(log cnt)。
 */

// P1776 数据范围：n <= 100, c[i] <= 10^5
// 每个物品拆分后的个数约 log2(10^5) ≈ 17 个
// 总物品数 m 大约 100 * 17 = 1700，开 2000 足够
const int MAXM = 2005;
const int MAXW = 40005;

int v[MAXM];      // 拆分后的物品价值
int w[MAXM];      // 拆分后的物品重量
int dp[MAXW];     // DP 数组
int n, t, m;      // n: 原物品数, t: 背包容量, m: 拆分后总物品数

/**
 * 01 背包空间压缩实现
 */
int compute() {
    // 初始化 DP 数组
    for (int j = 0; j <= t; j++) dp[j] = 0;

    for (int i = 1; i <= m; i++) {
        // 01 背包逆序遍历容量
        for (int j = t; j >= w[i]; j--) {
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }
    return dp[t];
}

int main() {
    // 提升 I/O 效率
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    while (cin >> n >> t) {
        m = 0; // 重置拆分后的物品计数器
        for (int i = 0; i < n; i++) {
            int value, weight, cnt;
            cin >> value >> weight >> cnt;

            /**
             * 二进制分组核心实现
             */
            for (int k = 1; k <= cnt; k <<= 1) {
                m++;
                v[m] = k * value;
                w[m] = k * weight;
                cnt -= k;
            }
            // 如果还有剩下的余数，单独作为一个物品
            if (cnt > 0) {
                m++;
                v[m] = cnt * value;
                w[m] = cnt * weight;
            }
        }
        
        cout << compute() << "\n";
    }

    return 0;
}
