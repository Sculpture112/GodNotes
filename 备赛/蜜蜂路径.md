# P2437 蜜蜂路线

## 题目描述

一只蜜蜂在下图所示的数字蜂房上爬动,已知它只能从标号小的蜂房爬到标号大的相邻蜂房,现在问你：蜜蜂从蜂房 $m$ 开始爬到蜂房 $n$，$m<n$，有多少种爬行路线？（备注：题面有误，右上角应为 $n-1$）

![](https://cdn.luogu.com.cn/upload/pic/1575.png)

## 输入格式

输入 $m,n$ 的值。

## 输出格式

爬行有多少种路线。

## 输入输出样例 #1

### 输入 #1

```
1 14
```

### 输出 #1

```
377
```

## 说明/提示

对于 $100\%$ 的数据，$1 \le m,n\le 1000$。
```c++
#include <bits/stdc++.h>

using namespace std;

// #define int long long

// const int N = ;

long long memo[1005];

bool visited[1005];

  

long long dfs(int k)

{

    if (k == 0 || k == 1)

        return 1;

  

    if (visited[k])

        return memo[k];

    memo[k] = dfs(k - 1) + dfs(k - 2);

    visited[k] = true;

  

    return memo[k];

}

  

void solve()

{

    int m, n;

    cin >> m >> n;

  

    int distance = n - m;

  

    if (distance <= 0)

    {

        if (distance = 0)

            cout << 1 << endl;

        else

            cout << 0 << endl;

    }

    else

    {

        cout << dfs(distance) << endl;

    }

}

signed main()

{

    ios::sync_with_stdio(false), cin.tie(0), cout.tie(0);

    solve();

    return 0;

}
```
