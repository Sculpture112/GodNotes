```cpp
旺仔哥哥经常在牛客上刷题，但有一天他突然忘记了登录密码（他没有绑定邮箱或手机）。
```

他记得密码是通过对原文字符串 ss 中的每个小写英文字母向后错位 nn 次得到的。字母 zz 向后错位一次会变为 aa，以此类推循环。
现给出原文字符串 ss 和错位次数 nn，请帮助旺仔哥哥计算出最终密码。

输入描述：
第一行输入一个整数 nn (1≦n≦100)(1≦n≦100)，表示错位次数。
第二行输入一个由小写英文字母组成的字符串 ss (1≦∣s∣≦103)(1≦∣s∣≦103)，表示原文字符串。

输出描述：
输出一个字符串，表示对 ss 中每个字符向后错位 nn 次后得到的密码字符串。

示例1

输入：
```cpp
1
abcdefg
```
复制

输出：
```cpp
bcdefgh
```
复制

说明：
```cpp
∙ ∙每个字母向后移动 11 次，"a"→"b""a"→"b"，"b"→"c""b"→"c"，…，"g"→"h""g"→"h"，得到 "bcdefgh""bcdefgh"。
```

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    int n;
    string s;

    // 读入位移次数和字符串
    if (!(cin >> n >> s)) return 0;

    for (char &c : s) {
        // 使用循环移位公式
        // 注意：先减去 'a' 变成 0-25，处理完再加回 'a'
        c = (c - 'a' + n) % 26 + 'a';
    }

    cout << s << endl;

    return 0;
}
```
