```cpp
# P1149 [NOIP 2008 提高组] 火柴棒等式## 题目描述给你 $n$ 根火柴棍，你可以拼出多少个形如 $A+B=C$ 的等式？等式中的 $A$、$B$、$C$ 是用火柴棍拼出的整数（若该数非零，则最高位不能是 $0$）。用火柴棍拼数字 $0\sim9$ 的拼法如图所示：![](https://cdn.luogu.com.cn/upload/image_hosting/p5hsawt2.png)注意：1. 加号与等号各自需要两根火柴棍；2. 如果 $A\neq B$，则 $A+B=C$ 与 $B+A=C$ 视为不同的等式（$A,B,C\geq0$）；3. $n$ 根火柴棍必须全部用上。## 输入格式一个整数 $n(1 \leq n\leq 24)$。## 输出格式一个整数，能拼成的不同等式的数目。## 输入输出样例 #1### 输入 #1```14```### 输出 #1```2```## 输入输出样例 #2### 输入 #2```18```### 输出 #2```9```## 说明/提示【输入输出样例 1 解释】$2$ 个等式为 $0+1=1$ 和 $1+0=1$。【输入输出样例 2 解释】$9$ 个等式为$0+4=4$、$0+11=11$、$1+10=11$、$2+2=4$、$2+7=9$、$4+0=4$、$7+2=9$、$10+1=11$、$11+0=11$。noip2008 提高第二题
```

```cpp
#include <iostream>
#include <vector>

using namespace std;

// 计算单个整数需要的火柴数
int getCost(int num) {
    if (num == 0) return 6;
    int cost[] = {6, 2, 5, 5, 4, 5, 6, 3, 7, 6};
    int total = 0;
    while (num > 0) {
        total += cost[num % 10];
        num /= 10;
    }
    return total;
}

int main() {
    int n;
    cin >> n;
    int target = n - 4;
    int count = 0;

    // 枚举 A 和 B，范围设定在 1000 左右足够覆盖 n=24
    for (int a = 0; a <= 1000; ++a) {
        for (int b = 0; b <= 1000; ++b) {
            int c = a + b;
            if (getCost(a) + getCost(b) + getCost(c) == target) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```
