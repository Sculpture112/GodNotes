# 定义学生结构体

> 现有 N 名同学参加了期末考试，并且获得了每名同学的信息：语文、数学、英语成绩（均为不超过 150 的自然数）。如果某对学生 ⟨i,j⟩ 的每一科成绩的分差都不大于 5，且总分分差不大于 10，那么这对学生就是“旗鼓相当的对手”。现在想知道这些同学中，有几对“旗鼓相当的对手”？同样一个人可能会和其他好几名同学结对。输入格式第一行一个正整数 N。接下来 N 行，每行三个整数，其中第 i 行表示第 i 名同学的语文、数学、英语成绩。最先读入的同学编号为 1。输出格式输出一个整数，表示“旗鼓相当的对手”的对数。输入输出样例输入 #1复制3
> 90 90 90
> 85 95 90
> 80 100 91

输出 #1复制

> 2

```cpp
#include <iostream>
#include <vector>
#include <cmath> // 用于 abs() 函数取绝对值

using namespace std;

// 定义学生结构体
struct Student {
    int chinese;
    int math;
    int english;
    int total;
};

int main() {
    int n;
    if (!(cin >> n)) return 0;

    vector<Student> stu(n);
    // 1. 读入所有学生信息
    for (int i = 0; i < n; i++) {
        cin >> stu[i].chinese >> stu[i].math >> stu[i].english;
        // 计算总分
        stu[i].total = stu[i].chinese + stu[i].math + stu[i].english;
    }

    int count = 0;
    // 2. 两两比较 (i 和 j)
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            // 判定条件1：单科分差 <= 5
            bool c1 = abs(stu[i].chinese - stu[j].chinese) <= 5;
            bool c2 = abs(stu[i].math - stu[j].math) <= 5;
            bool c3 = abs(stu[i].english - stu[j].english) <= 5;

            // 判定条件2：总分分差 <= 10
            bool c4 = abs(stu[i].total - stu[j].total) <= 10;

            // 如果全部满足，则是旗鼓相当的对手
            if (c1 && c2 && c3 && c4) {
                count++;
            }
        }
    }

    // 3. 输出结果
    cout << count << endl;

    return 0;
}
```
