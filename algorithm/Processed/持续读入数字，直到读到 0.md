# 持续读入数字，直到读到 0

> 小鱼最近被要求参加一个数字游戏，要求它把看到的一串数字 ai​（长度不一定，以 0 结束），记住了然后反着念出来（表示结束的数字 0 就不要念出来了）。这对小鱼的那点记忆力来说实在是太难了，你也不想想小鱼的整个脑袋才多大，其中一部分还是好吃的肉！所以请你帮小鱼编程解决这个问题。输入格式一行内输入一串整数，以 0 结束，以空格间隔。输出格式一行内倒着输出这一串整数，以空格间隔。输入输出样例输入 #1复制运行3 65 23 5 34 1 30 0

输出 #1复制运行

> 30 1 34 5 23 65 3

```cpp
#include <iostream>
#include <vector>

using namespace std;

int main() {
    int num;
    vector<int> a; // 创建一个动态数组来存储数字

    // 持续读入数字，直到读到 0
    while (cin >> num && num != 0) {
        a.push_back(num); // 把数字放到数组末尾
    }

    // 从数组的最后一个元素开始往前遍历
    // a.size() 获取数组长度
    for (int i = a.size() - 1; i >= 0; i--) {
        cout << a[i];

        // 如果不是最后一个输出的数字，就补一个空格
        if (i > 0) {
            cout << " ";
        }
    }

    return 0;
}
```
