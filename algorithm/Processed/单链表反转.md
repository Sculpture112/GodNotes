# 单链表反转

```cpp
// 单链表反转
ListNode* reverseList(ListNode* head) {
    ListNode* pre = nullptr;
    ListNode* next = nullptr;
    while (head != nullptr) {
        next = head->next; // 1. 记录
        head->next = pre;  // 2. 反转
        pre = head;        // 3. 推进
        head = next;       // 4. 移动
    }
    return pre;
}
```
