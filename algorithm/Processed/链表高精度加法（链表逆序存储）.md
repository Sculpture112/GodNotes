# 链表高精度加法（链表逆序存储）

```cpp
// 链表高精度加法（链表逆序存储）
static ListNode* addTwoNumbers(ListNode* h1, ListNode* h2) {
    ListNode* ans = nullptr; // 结果头节点
    ListNode* cur = nullptr; // 结果尾指针
    int carry = 0;           // 进位

    // 循环条件：只要 h1 或 h2 还有节点未处理，就继续
    for (int sum, val; 
         h1 != nullptr || h2 != nullptr; 
         h1 = (h1 == nullptr ? nullptr : h1->next), 
         h2 = (h2 == nullptr ? nullptr : h2->next)) {

        // 1. 求和：节点值 + 节点值 + 进位
        sum = (h1 == nullptr ? 0 : h1->val)
            + (h2 == nullptr ? 0 : h2->val)
            + carry;

        val = sum % 10;   // 当前位保留的数值
        carry = sum / 10; // 产生的新进位

        // 2. 创建新节点并连接
        if (ans == nullptr) {
            ans = new ListNode(val);
            cur = ans;
        } else {
            cur->next = new ListNode(val);
            cur = cur->next;
        }
    }

    // 3. 收尾：如果最后还有进位，补一个节点
    if (carry == 1) {
        cur->next = new ListNode(1);
    }

    return ans;
}
```
