## 题目描述

有 $N$ 条绳子，它们的长度分别为 $L_i$。如果从它们中切割出 $K$ 条长度相同的绳子，这 $K$ 条绳子每条最长能有多长？答案保留到小数点后 $2$ 位(直接舍掉 $2$ 位后的小数)。

## 输入格式

第一行两个整数 $N$ 和 $K$，接下来 $N$ 行，描述了每条绳子的长度 $L_i$ 。

## 输出格式

切割后每条绳子的最大长度。答案与标准答案误差不超过 $0.01$ 或者相对误差不超过 $1\%$ 即可通过。

## 输入输出样例 #1

### 输入 #1

```
4 11
8.02
7.43
4.57
5.39
```

### 输出 #1

```
2.00
```

## 说明/提示

对于 $100\%$ 的数据 $0<L_i\leq 100000.00,0<n\leq 10000,0<k\leq 10000$

#include <bits/stdc++.h>

using namespace std;

// #define int long long

void solve()

{

    int n, k;

    cin >> n >> k;

    vector<double> repos(n);

    double ans = 0;

    for (int i = 0; i < n; i++)

    {

        cin >> repos[i];

    }

  

    double left = 0.0, right = 100000.0;

    for (int i = 0; i < 100; i++)

    {

        double mid = left + (right - left) / 2;

        int count = 0;

        for (int j = 0; j < n; j++)

        {

            count += (int)(repos[j] / mid);

        }

        if (count >= k)

        {

            left = mid;

            ans = mid;

        }

        else

        {

            right = mid;

        }

    }

    cout << fixed << setprecision(2)<< ans  << endl;

}

signed main()

{

    ios::sync_with_stdio(false), cin.tie(0), cout.tie(0);

    solve();

    return 0;

}
