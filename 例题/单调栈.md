[[已AC]]  [[例题/单调栈|单调栈]]

有 NN 个人排成一排，从左到右依次编号。排在第 ii 位的人叫做第 ii 号人。第 ii 号人的身高是 AiAi​。

对于每个 i=1,2,…,Ni=1,2,…,N，判断在第 ii 号人的左边，是否存在比第 ii 号人高的人。如果有，找出离第 ii 号人最近的那个人。

### 约束条件

- 1≤N≤1001≤N≤100
- 1≤Ai≤1001≤Ai​≤100
- 所有输入值均为整数。

### 输入格式

输入从标准输入读取，格式如下：

NN
A1A1​ A2A2​ …… ANAN​

### 输出格式

输出 NN 行。

第 ii 行如果第 ii 号人的左边没有比第 ii 号人高的人，输出 −1−1；否则输出距离第 ii 号人最近的那个人的编号。

### 样例 1

|Inputcopy|Outputcopy|
|---|---|
|4<br>4 3 2 5|-1<br>1<br>2<br>-1|

- 第 11 号人左边没人，所以第一行输出 −1−1。
- 第 22 号人左边只有第 11 号人比第 22 号人高，所以第二行输出 11。
- 第 33 号人左边有几个人比第 33 号人高，离第 33 号人最近的是第 22 号人，所以第三行输出 22。
- 第 44 号人左边没有比第 44 号人高的人，所以第四行输出 −1−1。

### 样例 2

|Inputcopy|Outputcopy|
|---|---|
|3<br>7 7 7|-1<br>-1<br>-1|

可能会有多个人身高相同。

### 样例 3

|Inputcopy|Outputcopy|
|---|---|
|6<br>31 9 17 10 2 9|-1<br>1<br>1<br>3<br>4<br>4|
```cpp
// 假设 arr 是身高数组，n 是人数
// ans 数组用来存每个人的答案
int st[105]; // 栈，存的是下标
int r = 0;   // 栈顶指针

for (int i = 0; i < n; i++) {
    // 1. 踢人阶段：我是大个子，我把左边比我矮或者跟我一样高的人都踢掉
    // 因为对于未来的元素来说，我“又高又近”，被踢掉的人没用了
    while (r > 0 && arr[st[r - 1]] <= arr[i]) {
        r--; 
    }
    
    // 2. 结算阶段（只看左边）：踢完之后，栈顶剩下的就是左边离我最近且比我高的人
    if (r > 0) {
        ans[i] = st[r - 1] + 1; // 题目通常要求编号从 1 开始，所以下标 + 1
    } else {
        ans[i] = -1; // 栈空了，说明左边没人比我高
    }
    
    // 3. 入栈阶段：我自己入栈，等待被右边的人审视
    st[r++] = i;
}

```
