### 题目名称：C. 巨大的苹果堆 (Huge Pile)

**题目描述**

Andrei 有一堆含有 $n$ 个苹果的巨大苹果堆。他可以将一个苹果堆平分为两个较小的堆：如果当前堆有 $x$ 个苹果，他可以将其分为大小为 $\lfloor \frac{x}{2} \rfloor$（下取整）和 $\lceil \frac{x}{2} \rceil$（上取整）的两堆。

每次这样的操作（将一堆分成两堆）需要花费 Andrei **1 分钟**的时间。

Andrei 想要吃掉 $k$ 个苹果，但他完全不想一个一个去数。因此，他希望通过不断的拆分，最终获得一个**恰好包含 $k$ 个苹果**的堆。

你的任务是判断是否可能通过拆分操作获得大小正好为 $k$ 的堆。如果可以，请找出获得该堆所需的**最小分钟数**。

---

**输入格式**

- 第一行包含一个整数 $t$ ($1 \le t \le 10^4$)，表示测试用例的数量。
    
- 接下来的每一行包含两个整数 $n$ 和 $k$ ($1 \le n, k \le 10^9$)，分别代表初始苹果堆的数量和 Andrei 目标堆的数量。
    

**输出格式**

- 对于每个测试用例，如果无法获得恰好为 $k$ 个苹果的堆，输出 `-1`。
    
- 否则，输出获得该堆所需的最小操作次数（分钟数）。
    

---

**样例解释**

1. **输入 `10 3`，输出 `2`**：
    
    - 第 1 分钟：将 10 分为 (5, 5)。
        
    - 第 2 分钟：将其中一个 5 分为 (2, 3)。成功获得大小为 3 的堆。
        
2. **输入 `11 5`，输出 `1`**：
    
    - 第 1 分钟：将 11 分为 (5, 6)。成功获得大小为 5 的堆。
        
3. **输入 `21 4`，输出 `-1`**：
    
    - 无论如何划分，只能得到大小为 1, 2, 3, 5, 6, 10, 11 或 21 的堆，无法得到 4。
        
4. **输入 `1000000000 1`，输出 `29`**：
    
    - 通过不断二分，最少需要 29 次操作才能分出大小为 1 的堆。

```c++
#include <iostream>
#include <set>
#include <vector>

using namespace std;

/**
 * 解决思路：
 * 每一层代表一次操作（1分钟）。
 * 初始层只有 {n}。
 * 每一层通过对上一层的每个元素进行 floor(x/2) 和 ceil(x/2) 得到新的一层。
 * 如果在某一层中发现了 k，则当前的层数即为最小时间。
 */
void solve() {
    long long n, k;
    cin >> n >> k;

    if (n == k) {
        cout << 0 << endl;
        return;
    }

    set<long long> current_layer;
    current_layer.insert(n);
    int minutes = 0;

    // 因为 n 最大 10^9，最多 30 层左右就会全部变成 1 或 0
    while (!current_layer.empty()) {
        minutes++;
        set<long long> next_layer;
        bool found = false;

        for (long long x : current_layer) {
            if (x < 2) continue; // 无法再分

            long long low = x / 2;
            long long high = (x + 1) / 2;

            if (low == k || high == k) {
                found = true;
                break;
            }

            if (low > k) next_layer.insert(low);
            if (high > k) next_layer.insert(high);
        }

        if (found) {
            cout << minutes << endl;
            return;
        }

        if (next_layer.empty()) break;
        current_layer = next_layer;
    }

    cout << -1 << endl;
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int t;
    cin >> t;
    while (t--) {
        solve();
    }
    return 0;
}
```
