## Vim 快捷键速查表 (Vim Cheat Sheet)

Vim 是 Vi 的现代版本，是 Unix 系统中备受推崇的文本编辑器 。在 Vim 中，你可以通过组合不同的**导航**和**命令**来实现强大的编辑功能 。

+1

### 1. 基础模式

- **命令模式 (Command Mode)**：用于导航和修改 。
    
- **输入/插入模式 (Write/Insert Mode)**：用于输入文本 。
    
- **命令行模式 (Command Line Mode)**：执行保存、退出等指令 。
    

---

### 2. 导航指令 (Navigation)

| **按键**           | **功能说明**       |
| ---------------- | -------------- |
| **$w$ / $b$**    | 向前/向后移动一个单词    |
| **$W$ / $B$**    | 向前/向后移动一个完整单词  |
| **$e$ / $E$**    | 移动到单词结尾        |
| **$fx$**         | 在当前行向前搜索字符 "x" |
| **$($**          | 跳转到句首          |
| **$)$**          | 跳转到句尾          |
| **$0$ / $\$$**   | 跳转到行首/行尾       |
| **$\{$**         | 跳转到段落开头        |
| **$\}$**         | 跳转到段落结尾        |
| **$/ab$**        | 向前搜索字符串 "ab"   |
| **$ma$**         | 将当前位置标记为 "a"   |
| **$`a$**         | 跳转回标记 "a" 的位置  |
| **$Ctrl-b / f$** | 向上/向下翻页        |

---

### 3. 编辑与组合命令 (Modification & Combined)

|**组合键**|**功能说明**|
|---|---|
|**$x$**|删除光标处的字符|
|**$r$**|替换光标处的单个字符|
|**$R$**|连续替换多个字符|
|**$u$**|撤销 (Undo)|
|**$Ctrl-r$**|重做 (Redo)|
|**$J$**|合并两行|
|**$\sim$**|在光标处切换大小写|
|**$cc$**|修改整行|
|**$cw$ / $ce$**|修改单词 / 修改至词尾|
|**$dd$**|删除整行|
|**$dw$ / $de$**|删除单词 / 删除至词尾<br><br>+1|
|**$yy$**|复制（抽出）整行|
|**$yw$**|复制单词|
|**$p$**|粘贴已复制或删除的文本|

---

### 4. 常用系统指令

- **$:w$**：保存文件
    
- **$:q$**：退出 Vim
    
- **$Esc$**：退出当前模式，返回命令模式
    

> **说明**：`C-x` 是 `Ctrl-x` 的简写 。
